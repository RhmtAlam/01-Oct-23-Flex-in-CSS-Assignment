The CSS Flexbox layout Model

flexbox was designed for layout in one dimension - either a row or a column.

The CSS Grid Layout Model.

Grid was designed for two-dimensional layout - rows, and columns at the same time.

When to use which One Flex model or Grid Model in CSS

No i will not choose to use one over the other but in some cases i will choose as follows.
If you are using flexbox and find yourself disabling some of the flexibility, you probably need to use CSS Grid Layout.
An example would be if you are setting a percentage width on a flex item to make it line up with other items in a row above. In that case, a grid is likely to be a better choice
